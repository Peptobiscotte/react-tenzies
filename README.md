# React Tenzies 
Tenzies dice game made using React

## How to run :
+ npm install
+ npm run dev

The web server should be available at http://localhost:5173/react-tenzies/

## Stack :
+ Vite to launch and configure local server and to bundle source files.
+ React to design components.
+ JS
+ CSS vanilla
+ HTML5
